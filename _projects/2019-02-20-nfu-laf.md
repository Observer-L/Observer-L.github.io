## 介绍

南苑寻物（南寻）是一款**以地图为载体、服务于中山大学南方学院的失物招领**小程序。
解决了在校师生失物招领无平台、低效不便的问题，为在校师生**提供一个更加高效便捷的失物招领平台**。

----

## 功能特性：
- [x] 地图热点信息一览
- [x] 发布交互方便快捷
- [x] 一键联系
- [x] 失物招领历史记录
- [ ] 隐私保护相关设置
- [ ] 更多的物品OCR识别支持
